## 研究要点

- 研究纳入1,246名社区受试者，分析小米可穿戴设备测量的PWV数据
- 发现PWV与年龄、血压等传统风险因素显著相关
- 构建基于XGBoost的预测模型，可有效预测PWV超标风险(AUC=0.89)、高血压风险(AUC=0.85)和综合心血管风险(AUC=0.83)
- 证实便携式可穿戴设备在动脉硬化早期筛查和心血管风险评估中的应用价值
